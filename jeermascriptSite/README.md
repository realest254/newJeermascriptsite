# Jeermascript Agency Website

This repository contains the source code for the Jeermascript Agency website, a digital marketing agency specializing in SEO, social media, and web development services.

## Features

- React 18 for efficient UI rendering
- React Router for navigation
- React Helmet for SEO optimization
- Vite for fast builds and development
- Sitemap generation for improved SEO
- GitHub Pages integration for easy deployment

## Project Structure

- `src/`: Source files

  - `routes.js`: Route definitions
- `public/`: Static assets
- `generateSitemap.js`: Sitemap generation script
- `vite.config.js`: Vite configuration
